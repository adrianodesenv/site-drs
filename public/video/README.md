# Vídeo Hero

Coloque o vídeo do hero nesta pasta.

## Formato recomendado:
- **hero-video.mp4** - Formato MP4 (H.264) para compatibilidade máxima
- **hero-video.webm** - Formato WebM (opcional, para melhor compressão)

## Especificações recomendadas:
- Resolução: 1920x1920px (quadrado) ou proporção 1:1
- Codec: H.264 para MP4, VP9 para WebM
- Taxa de bits: 2-5 Mbps
- Duração: Loop curto (5-15 segundos recomendado)

O vídeo será exibido automaticamente em loop, sem som, no componente Hero.
